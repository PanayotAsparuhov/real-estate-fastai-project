# Анализ на големи данни за недвижими имоти

Демонстрационен проект за прогнозиране на цена на недвижим имот чрез FastAI и tabular data.

## Описание на проекта

Целта на проекта е да се изгради модел за прогнозиране на цена на недвижим имот по таблични данни.

Използван е dataset `Housing.csv`, съдържащ 545 записа и 13 признака, описващи характеристики на жилищни имоти като площ, брой спални, брой бани, етажност, наличие на климатик, паркоместа, предпочитана зона и степен на обзавеждане.

## Използван dataset

За разработката е използван dataset `Housing.csv`, съдържащ таблични данни за недвижими имоти.  
Dataset-ът включва 545 наблюдения и 13 признака.

## Използван модел

Като основен модел е използван `FastAI tabular_learner`, приложен върху regression задача с цел прогнозиране на цената на имот.

Данните са предварително обработени чрез:
- категориализация (`Categorify`)
- попълване на липсващи стойности (`FillMissing`)
- нормализация (`Normalize`)

За подобряване на обучението е приложена логаритмична трансформация на целевата променлива, а крайните прогнози са преобразувани обратно към реалната стойност на цената.

## Резултат

След обучението на модела са изчислени метрики за оценка на качеството като:
- MAE
- RMSE
- R²

Получените резултати показват, че моделът е демонстрационен и има ограничения по отношение на точността, но успешно илюстрира процеса по:
- подготовка на tabular data
- обучение на модел
- оценка на резултатите
- извличане на примерна прогноза
